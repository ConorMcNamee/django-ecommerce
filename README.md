# django-ecommerce

Simple django E-Commerce website using:
  - Django 
  - Django REST Framework
  - Pillow
  - React

To run the backend:

  `python manage.py makemigrations`,
  
  `python manage.py migrate`,
  
  `python manage.py runserver`,
  
To Run the Frontend:

  `npm run build`,
 
  `npm install `,
  
  `npm start`,
